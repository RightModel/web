# Roadmap

rightmodel.dev is built with a focused, utility-first approach. We track progress against specific milestones rather than dates.

## Currently Active
- **v2 Build**: Core product launch implementing the Precomputed AI (PAI) architecture.
- **Provider Expansion**: Add tier mapping for additional high-value OpenRouter models as they gain adoption.

## Short-term (Next 3 Months)
- **Task Page Expansion**: Increase indexed task archetypes from 50 to 100 based on organic search demand and user submissions.
- **Ruleset Refinement**: Improve routine/moderate classification accuracy for specific domains (e.g., SQL generation, Kubernetes debugging).

## Long-term / Future
- **Agentic Readiness**: Explore exposing an MCP server to allow autonomous agents direct access to the rightmodel heuristic.
- **Platform Integrity**: Maintain the daily pricing cache and ensure tier mappings remain accurate as the model landscape shifts.
